# To-Do List App

## Overview
A web-based To-Do List application built with HTML, CSS, and JavaScript for 
CSE 310 Applied Programming Module 3 at BYU-Idaho. The app allows users to 
add tasks, mark them as completed, filter tasks by status, and delete tasks 
with a confirmation dialog. All functionality runs directly in the browser.

## Demo Video
[Watch the demo on YouTube](https://youtu.be/npP2BoAVqwg)

## Features
- Add new tasks using an input field or by pressing Enter
- Mark tasks as completed with a strikethrough effect
- Filter tasks by All, Active, or Completed
- Delete tasks with a confirmation dialog
- Live statistics showing total, completed, and remaining tasks
- Dynamic DOM updates without reloading the page

## JavaScript Concepts Demonstrated
- DOM manipulation (creating, updating, and removing elements dynamically)
- ES6 Array functions: find, filter, and forEach
- Recursion (recursive function to count completed tasks)
- Third-party JavaScript library integration (SweetAlert2 via CDN)
- DOM structures styled with CSS

## How to Run
1. Download or clone this repository
2. Open the project folder
3. Double-click `index.html` to open it in your web browser
4. Start adding and managing tasks

## What I Learned
- How to manipulate the DOM dynamically using JavaScript
- How to use ES6 array methods like find, filter, and forEach to manage data
- How recursion works by defining a base case and a recursive case
- How to integrate a third-party library using a CDN link
- How to build an interactive, responsive web application from scratch
